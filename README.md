# E-Tutoria Landing Page

This project is a static landing page built using **HTML and CSS**.
The goal of the project was to practice building structured layouts and responsive design using basic frontend technologies.

## 🚀 Technologies

* HTML5
* CSS3
* Flexbox

## ⚙️ Features

* Multi-section landing page
* Structured and semantic HTML
* Clean UI layout

## 🎯 Purpose

This project was created as part of a **HTML/CSS Basic practice project** to improve layout and styling skills.
